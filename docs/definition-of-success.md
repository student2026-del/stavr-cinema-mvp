# Definition of Success для StavrCinema

## Критерії готовності (Що зроблено / Output):
* [ ] Є репозиторій з описом (README).
* [ ] Готова верстка екрану вибору фільму.
* [ ] Готова верстка екрану залу (місця натискаються).

## Критерії перевірки (Як працює / Outcome):
* 3 з 3 тестових користувачів змогли «купити» квиток без підказки.
* Ніхто не заплутався у схемі залу.
* Час проходження сценарію — до 2 хвилин.

## Управління невизначеністю (Спрощення):
* Ми не підключаємо реальну оплату (тільки імітація).
* Ми не використовуємо базу даних (дані про фільми "зашиті" в код).
