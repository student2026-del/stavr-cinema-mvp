# StavrCinema - Система онлайн-бронювання квитків

Даний репозиторій містить вихідний код та документацію навчального проєкту **StavrCinema**. Метою проєкту є розробка веб-сервісу для автоматизації процесу вибору місць та купівлі квитків у кінотеатрі.

## 1. Проблема та Пітч 
Відвідувачі кінотеатрів часто стикаються з чергами біля кас та обмеженим вибором місць безпосередньо перед сеансом. **StavrCinema** - це веб-застосунок, що дозволяє користувачам заздалегідь переглянути розклад, обрати зручні місця на схемі залу та здійснити бронювання онлайн, що суттєво економить час та покращує користувацький досвід.

## 2. Ядро цінності та MVP
Основна цінність продукту полягає у спрощенні процесу купівлі квитка. У рамках першої ітерації реалізується лише критично важливий функціонал.

**Функціональні вимоги до MVP:**
* **Розклад сеансів:** Відображення списку доступних фільмів та часу сеансів.
* **Інтерактивна схема залу:** Візуалізація схеми місць із можливістю вибору вільних крісел.
* **Бронювання:** Механізм підтвердження вибору та імітація успішної покупки.

## 3. Стейкхолдери
* **Кінцевий користувач:** Зацікавлений у швидкому та інтуїтивно зрозумілому інтерфейсі для бронювання без черг.
* **Адміністрація кінотеатру:** Зацікавлена у підвищенні заповнюваності залів та оптимізації роботи касирів.
* **Розробник:** Відповідальний за технічну реалізацію проєкту в межах встановлених часових та ресурсних обмежень.

## 4. Критерії успіху 
Проєкт вважатиметься успішним за умови виконання наступних показників:
*  Веб-сайт коректно відображається на десктопних та мобільних пристроях.
*  Сценарій користувача (вибір фільму -> вибір місця -> підтвердження) виконується без помилок.
*  Час проходження повного сценарію бронювання не перевищує 2 хвилин.

## 5. План робіт 
* Ініціалізація репозиторію та налаштування оточення.
* Верстка головної сторінки (список фільмів).
* Реалізація візуальної схеми залу.
* Програмування логіки вибору місць (обробка подій click/hover).
* Реалізація модального вікна підтвердження замовлення.
