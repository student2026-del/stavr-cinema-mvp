# StavrCinema - Сервіс онлайн-бронювання квитків

## 1. Пітч
Любителі кіно часто витрачають час у чергах або стикаються з відсутністю хороших місць перед сеансом. **StavrCinema** - це вебсайт, який дозволяє користувачам заздалегідь вибрати фільм, забронювати зручні місця та купити квиток онлайн, гарантуючи комфортний похід у кіно без зайвого очікування.

## 2. Ядро цінності та MVP
Ми не робимо складний портал з новинами. Наша мета - **простота покупки**.
**Функціонал MVP:**
* Перегляд розкладу сеансів.
* Інтерактивна схема залу для вибору місця.
* Вікно підтвердження бронювання («Квиток куплено»).

## 3. Стейкхолдери 
* **Відвідувачі:** Хочуть швидко купити квиток і не стояти в касі.
* **Адміністрація кінотеатру:** Хоче бачити заповнені зали.
* **Розробник:** Хоче здати проєкт вчасно, використовуючи прості технології.

## 4. Ознаки успіху 
Ми вважатимемо, що перша версія успішна, якщо:
1.  Користувач може відкрити сайт з телефону.
2.  Можна вибрати місце на схемі та натиснути «Купити» без помилок.
3.  Весь процес займає менше ніж 2 хвилини.
